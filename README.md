# **_Lazy Traveler_**

### **Features:**

- Hotel booking management app. Single page web application.
- Customers can select the origin and destination of their trip.
- Registered users can see hotels based on their destination. And can book a hotel.

### **Technology used:**

- React.js
- React Router
- Bootstrap
- React Hook Form
- Firebase auth
